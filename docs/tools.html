🛠️ Developer Batch Scripts (.bat)

Semua file shortcut tersedia di root folder proyek. Double-click untuk menjalankan. Gunakan CMD bukan PowerShell jika ada error "scripts is disabled".

Nama File Fungsi Kapan Digunakan
Install manual awal.bat ⭐ Setup lengkap otomatis: install npm + flutter + prisma db push + generate + seeding semua data Pertama kali setup di komputer baru
start_all.bat Buka 4 terminal sekaligus: Server (4000) + Dashboard Merchant (5173) + Admin (5174) + Distributor (5175) Setiap hari development
start_server.bat Jalankan hanya backend API di port 4000 dengan npm run dev (nodemon) Saat hanya butuh backend / testing API
seed_db.bat Isi ulang data demo: akun (merchant + admin), menu navigasi, produk PPOB Jika perlu reset data demo saja
update_db.bat Sinkronisasi schema.prisma ke PostgreSQL + generate Prisma Client Setiap kali ada perubahan schema database
build_apk.bat Build APK Release dengan pilihan interaktif: Rana Merchant, Rana Buyer, atau keduanya. Output di builds/ Deploy APK ke HP tanpa Android Studio
run_test.bat Jalankan unit test di sisi server Node.js Sebelum commit perubahan besar
💡 Alur kerja pertama kali:
1. Edit server/.env → 2. Buat database rana_pos di pgAdmin → 3. Double-click Install manual awal.bat → 4. Double-click start_all.bat
